To Bool
=========

A small library that converts typical "boolean" values to boolean

## Installation

  `npm install @gin93r/to-bool`

## Usage

    var toBool = require('@gin93r/to-bool');

    var b = toBool(0);
  
  
  Output should be `false`


## Tests

  `npm test`
